# 实时数据交换（RTDE）中文版指南-说明书下载

## 简介

本指南详细介绍了如何使用优傲控制器的数据同步协议，适用于在为优傲机器人开发URCaps/UR+时进行数据同步。本文档所提供的内容适用于CB3软件版本3.4及以上，以及所有转发系列软件版本。请注意，较新的软件版本可能行为会有所不同。

## 内容概述

本文档介绍了实时数据交换（RTDE）接口，该接口提供了一种通过标准TCP/IP连接将外部应用程序与UR控制器同步的方法，同时不会破坏UR控制器的任何实时属性。RTDE功能对于以下应用场景非常实用：

- 与现场总线驱动器（例如以太网/IP）交互
- 操作机器人I/O
- 绘制机器人状态（例如机器人轨迹）

默认情况下，RTDE接口在UR控制器运行时始终可用。同步配置灵活，可以涉及以下数据类型：

- 输出：机器人、关节、工具和安全状态、模拟和数字I/O以及通用输出寄存器
- 输入：数字和模拟输出以及通用输入寄存器

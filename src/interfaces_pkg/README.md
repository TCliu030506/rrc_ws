# msg_codec
This repository contains two packages, `msg_codec_cpp` and `msg_codec_py`, which provide message encoding and decoding functionalities for structured messages based on predefined command types and field types.

## msg_codec_cpp
This package provides message encoding and decoding functionalities for structured messages based on predefined command types and field types.
### Features
- 将字典数据编码为std_msgs/Uint8MultiArray消息.
- 将std_msgs/Uint8MultiArray消息解码为字典数据.

## msg_codec_py
This package provides message encoding and decoding functionalities for structured messages based on predefined command types and field types.
### Features
- 将字典数据编码为std_msgs/Uint8MultiArray消息.
- 将std_msgs/Uint8MultiArray消息解码为字典数据.
